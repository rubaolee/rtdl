# Goal 117: v0.2 Feature Usage Surface

Date: 2026-04-06
Status: accepted

## Purpose

Make the first v0.2 feature line easier to understand from a user point of
view.

This goal is not another backend or correctness goal. It takes the already
closed `segment_polygon_hitcount` feature family and presents it as a real user
workflow.

## Required outcomes

1. one app-style example exists for the feature
2. one short current-status note exists explaining what v0.2 now does beyond
   the archived v0.1 baseline
3. the workload cookbook points readers to the new example

## Honesty boundary

This goal must not claim any new backend maturity.

It only improves:

- explanation
- discoverability
- user-facing workflow clarity
