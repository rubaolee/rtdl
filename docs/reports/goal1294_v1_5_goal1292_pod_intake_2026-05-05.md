## Scale Adjustment

The accepted packet originally used:

```bash
PYTHONPATH=src:. python3 scripts/goal1292_v1_5_generic_optix_evidence_runner.py --copies 10000 --query-repeats 100
```

That command was interrupted after more than seven minutes at about 99 percent
CPU without producing `generic_optix_evidence.json`. The cause is the diagnostic
fixture shape: the CPU oracle/direct CPU path is O(rays*triangles), so 10,000
copies creates an unnecessarily large oracle workload for this evidence gate.

The primitive evidence was rerun at:

```bash
PYTHONPATH=src:. python3 scripts/goal1292_v1_5_generic_optix_evidence_runner.py --copies 256 --query-repeats 100
```

The packet generator has been updated to use this bounded primitive diagnostic
scale for future pod runs.
