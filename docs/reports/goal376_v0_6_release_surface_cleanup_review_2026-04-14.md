# Goal 376 Review: v0.6 release surface cleanup

## Decision

Accept.

## Why

- the new `v0.6` release package is coherent
- the front door still presents `v0.5.0` as the current released version
- `v0.6` is consistently described as release preparation rather than a tagged release
- the package shape now matches the existing release-package pattern in the repo
