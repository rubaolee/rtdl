# Goal 356: v0.6 Real Graph Dataset Preparation

## Why this goal exists

The bounded `v0.6` graph line now has working runtime and Linux evaluation
slices, but it still needs a real-data preparation path for larger tests.

## Scope

In scope:

- define the first real graph dataset candidates for `v0.6`
- add a bounded loader for SNAP-style edge lists
- keep the first real-data preparation path honest and narrow

Out of scope:

- downloading large datasets inside the repo
- full large-scale benchmark closure
- graph-language changes

## Exit condition

This goal is complete when the repo has:

- a saved real-dataset preparation report
- a saved external review
- a saved Codex consensus note
