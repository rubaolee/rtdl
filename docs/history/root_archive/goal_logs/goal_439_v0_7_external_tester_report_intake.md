# Goal 439: v0.7 External Tester Report Intake

## Goal

Create and maintain an external tester report intake gate for the continuing
`v0.7` DB line.

## Required Outcome

- Preserve each external tester report as a file-backed artifact.
- Maintain a triage ledger with:
  - report path
  - tester/source
  - date received
  - affected area
  - severity
  - finding summary
  - decision
  - mapped follow-up goal or waiver
- Classify each finding as:
  - release blocker
  - required follow-up before tag
  - non-blocking note
  - invalid / cannot reproduce
- Keep the `v0.7` no-tag/no-merge hold active unless all release blockers are
  resolved or explicitly waived with 2-AI consensus.

## Review Requirement

This goal requires at least 2-AI consensus before closure.
