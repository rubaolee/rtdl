# Legacy Learner Doc Version Notes

This page is the parking place for version-history context that used to appear
inside learner-facing docs.

The current learner path should describe the v2.0-facing RTDL surface directly:
Python application code, RTDL kernels, app-agnostic native engines, and
partner-owned columns for the Python+partner track. Older release details are
useful for project archaeology and evidence review, but they should not be in
the normal reader's first path.

## Where Older Version Information Lives

- v0.x and v1.x release packages: `docs/release_reports/`
- long-form goal evidence: `docs/reports/`
- external AI reviews: `docs/reviews/`
- project history index: `docs/history/README.md`
- complete history map: `history/COMPLETE_HISTORY.md`

Together, those files are the release-report archive for older learner-doc
version context.

## Current Public Reading Rule

Learner/user docs should present one coherent current surface. If a page needs
older release context, it should link here or to a specific release report
instead of embedding chronological release notes inline.
