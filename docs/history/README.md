# RTDL History Index

This directory is the user-facing doorway into RTDL history, release evidence,
review records, and project evolution notes.

The main docs intentionally describe RTDL as the current product. Historical
material lives here or in the linked evidence directories so new users do not
have to learn the project by reading its development timeline.

## What Belongs Here

Use this area for:

- release history,
- version evolution,
- goal and audit trails,
- review and consensus records,
- superseded design notes,
- evidence needed to understand why a current boundary exists.

Do not put beginner tutorial content here. Do not require new users to read
history before they can run RTDL.

## Current Documentation Rule

The front page, docs index, tutorial pages, and public documentation map should
explain the current RTDL user surface. They may link here for history, but they
should not embed project-evolution narratives, goal logs, or old release
timelines.
