# Goal 49: Live Documentation Rewrite

## Purpose

Review and rewrite the live project documentation so it is:

- correct
- consistent
- comprehensive
- easier to navigate

This goal does **not** rewrite history, logs, or archived per-goal reports.

## Rewrite Boundary

This goal targets:

- top-level current-state docs
- project vision/scope docs
- language and usage guides
- active dataset/workflow guidance

This goal does **not** target:

- `history/`
- `docs/reports/`
- archived goal execution reports
- historical closure notes

## Success Criteria

This goal is successful if it produces:

- an approved revision plan before editing
- a rewritten canonical live doc set
- a consistent navigation story across those docs
- Gemini review before edits
- Gemini review after edits
- final Codex/Gemini consensus on the result
