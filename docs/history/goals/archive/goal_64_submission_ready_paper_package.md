# Goal 64 Submission-Ready Paper Package

## Objective

Turn the existing RTDL RayJoin manuscript package into a cleaner
submission-ready paper package for external peer review.

This goal is not about changing the accepted experimental evidence. It is about
polishing and auditing the package that presents that evidence.

## Scope

Included:

- manuscript wording polish
- package README polish
- figure/table wording cleanup
- submission-boundary clarification
- final format/content audit by Codex and Gemini
- Claude review if Claude is operationally available

Excluded:

- new backend features
- new experiment families
- rewriting accepted historical reports
- changing the bounded evidence contract itself

## Acceptance

Goal 64 is complete only if:

1. the paper package is updated for clearer submission use
2. the manuscript still builds successfully
3. Codex reviews the package
4. Gemini reviews the package
5. Claude reviews the package if available, otherwise unavailability is
   recorded
6. final consensus is recorded before publication
