# Goal 194: v0.4 Content Package

## Purpose

Turn the `v0.4` direction decision into a concrete package the next version can
start from immediately.

This goal should answer:

- what `v0.4` is
- what `v0.4` is not
- which feature line is first
- what examples and docs must exist
- what acceptance criteria must be satisfied before release

## Scope

- define the release theme
- define the proposed public surfaces
- define non-goals
- define initial goal order
- define acceptance criteria and risks

## Constraint

The package must stay honest to the current repo state:

- RTDL is a non-graphical geometric-query runtime
- `v0.3.0` proved bounded application capability
- `v0.4` should convert that proof into public workload surface rather than
  into a demo-first milestone
