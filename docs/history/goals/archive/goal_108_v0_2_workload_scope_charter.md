# Goal 108: RTDL v0.2 Workload Scope Charter

Date: 2026-04-05
Status: complete

## Goal

Define the formal workload-scope charter for RTDL v0.2.

This charter is the gate required by Goal 107 before v0.2 workload expansion
can proceed.

## Required outputs

- one in-scope / experimental / out-of-scope matrix
- one rationale report
- one criticism/rebuttal pass
- one final accepted charter

## Acceptance

- every proposed v0.2 workload family is classified
- the charter explains why each family belongs where it does
- the charter includes entry criteria for new in-scope workloads
- criticism is recorded before acceptance
