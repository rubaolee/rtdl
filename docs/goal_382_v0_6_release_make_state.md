# Goal 382: v0.6 release-make state

## Why this goal exists

If Goal 381 accepts the `v0.6` package, the repo needs one final goal that
records the release-making state:
- front door
- docs index
- release package
- tag-preparation note

## Scope

In scope:

- update release-facing status from "under preparation" to the final accepted
  release state, if and only if Goal 381 accepts
- record the final package state needed for `v0.6.0`

Out of scope:

- post-release follow-up work
- new workload implementation

## Exit condition

This goal is complete when:

- the release-facing package language matches the final `v0.6.0` state
- the repo has a saved review note and Codex consensus note
- the release-making state is clear enough for tagging
