# RTDL v0.9.1 Release Package

This package records the released `v0.9.1` line.

`v0.9.1` releases the first bounded Apple RT backend slice: 3D
`ray_triangle_closest_hit` through Apple Metal/MPS on macOS Apple Silicon.

This release keeps the `v0.9.0` HIPRT and closest-hit support matrix intact and
adds Apple RT only for the narrow closest-hit path. It does not claim full Apple
backend parity, prepared Apple RT reuse, non-macOS support, or measured Apple
hardware speedup.
