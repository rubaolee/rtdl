# Goal 247: Archive Surface Audit Pass

## Objective

Seed the archive/report/history tier with a bounded representative audit pass
instead of trying to flatten the entire long-tail archive at once.

## Scope

This pass covers:

- preserved wiki drafts
- legacy or release-adjacent handoff files
- selected historically important reports that still sit close to the release
  audit trail

## Required Checks

- archive files are explicitly historical rather than silently acting as live docs
- easy-to-reach archive files do not contain broken maintainer-local absolute links when low-risk cleanup is available
- stale but historically valid files are marked as such instead of being treated as product-facing defects
- higher-noise archive files are classified for future cleanup rather than rewritten blindly
