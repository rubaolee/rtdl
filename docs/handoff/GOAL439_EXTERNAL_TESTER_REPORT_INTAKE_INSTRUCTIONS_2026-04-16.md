# Goal 439 External Tester Report Intake Instructions

Please save each external tester report as a file and provide the full absolute
path to Codex.

Preferred report format:

- tester/source name
- host OS and machine
- git branch / commit if known
- exact commands run
- full stdout/stderr or JSON result path
- expected result
- actual result
- whether this appears to be correctness, performance, docs, install, API, or
  release-claim related
- any local patch or workaround used

Codex will triage each report into:

- `S0 blocker`
- `S1 required`
- `S2 follow-up`
- `S3 note`
- `invalid`
