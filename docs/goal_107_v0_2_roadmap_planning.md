# Goal 107: RTDL v0.2 Roadmap Planning

Date: 2026-04-05
Status: complete

## Goal

Define a credible RTDL v0.2 roadmap that is ambitious enough to be worth
pursuing and disciplined enough to remain achievable.

## Required outputs

- one concrete v0.2 roadmap
- one adversarial critique/rebuttal report
- one recommended goal ordering for the first v0.2 wave
- independent AI participation in the planning review
- final consensus on whether the roadmap is promising or still too diffuse

## Core planning questions

1. Which additional workloads are truly in RTDL scope?
2. What does a pure code-generation RTDL product mode mean?
3. How much performance work should still focus on NVIDIA RT cores in v0.2?
4. Which ideas are promising, and which should be explicitly deferred?

## Acceptance

- the roadmap names a small set of real v0.2 pillars
- scope creep is explicitly resisted
- criticism and rebuttal are recorded, not hidden
- the final package states whether v0.2 is promising under the chosen scope
