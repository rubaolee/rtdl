# Archived Example Helpers

This directory holds runnable compatibility helpers that are still imported by
current unified examples or retained for audit tests.

They are not first-run examples. New readers should start from:

- `../../README.md`
- `../../rtdl_database_analytics_app.py`
- `../../rtdl_apple_rt_demo_app.py`

Current public wrappers:

| Public wrapper | Archived helpers |
| --- | --- |
| `rtdl_database_analytics_app.py` | `rtdl_v0_7_db_app_demo.py`, `rtdl_v0_7_db_kernel_app_demo.py` |
| `rtdl_apple_rt_demo_app.py` | `rtdl_apple_rt_closest_hit.py`, `rtdl_apple_rt_visibility_count.py` |
