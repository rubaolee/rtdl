# Spatial / RayJoin Study

This directory contains the RTDL v2.0 spatial join application used for
RayJoin-inspired evaluation.

| File | Purpose |
| --- | --- |
| `rtdl_rayjoin_v2_spatial_join_app.py` | grouped ray/shape spatial join program over RTDL v2.0 |

Boundary: this is a serious RTDL implementation study, not a claim that RTDL
reproduces every result or optimization from the RayJoin paper.
