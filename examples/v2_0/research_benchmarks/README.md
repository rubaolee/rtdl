# Research Benchmarks

This directory holds serious performance studies that compare RTDL v2.0
programs with paper-style or optimized external baselines.

These are not first-run tutorials. Read each subdirectory README and its linked
reports before interpreting timings.

| Directory | Study |
| --- | --- |
| `hausdorff_xhd/` | Hausdorff distance study informed by X-HD techniques |
| `spatial_rayjoin/` | spatial join study informed by RayJoin |
