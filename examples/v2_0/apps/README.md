# Application Examples

These are complete v2.0 examples that combine Python control logic, RTDL
primitive calls, and partner continuation where appropriate.

| Directory | App family |
| --- | --- |
| `analytics/` | graph and database analytics wrappers |
| `geospatial/` | service coverage, hotspots, facility assignment, road hazard, sales-risk examples |
| `ml/` | ANN candidates, outlier detection, DBSCAN-style clustering |
| `robotics/` | robot collision screening |
| `simulation/` | Barnes-Hut-style candidate discovery |
| `trajectory/` | continuous Frechet distance application |
