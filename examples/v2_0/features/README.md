# Feature Examples

These examples are grouped by RTDL feature family. They are smaller than full
apps and are the best place to learn one primitive or output contract at a
time.

| Directory | Feature family |
| --- | --- |
| `ray_queries/` | ray/triangle any-hit, visibility rows, row reductions |
| `neighbors/` | fixed-radius neighbors and KNN rows |
| `database/` | columnar scans and grouped reductions |
| `graph/` | frontier traversal and triangle counting |
| `spatial/` | segment/polygon rows, overlap summaries, Jaccard summaries |
