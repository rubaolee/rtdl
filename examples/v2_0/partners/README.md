# Partner Continuation Examples

These examples show how a user can continue RTDL outputs with partner-owned
arrays or user-owned native code. They are advanced examples, not a claim that
RTDL accelerates arbitrary partner framework programs.

| File | Purpose |
| --- | --- |
| `rtdl_partner_anyhit.py` | minimal Python+partner column handoff |
| `rtdl_control_apps_cupy_rawkernel.py` | CuPy RawKernel continuations for selected app-level summaries |
| `rtdl_hausdorff_user_cpp_continuation.py` | user-owned C++ continuation around RTDL nearest-neighbor rows |
