# Generated Examples

This directory contains preserved generated RTDL output bundles and generated
program examples.

These files are useful when you want to inspect what the generate-only flow
produces or hand a generated bundle to another person or tool.

They are **not** the main first-run entry points for new users.

If you are starting fresh, go first to:

- `../README.md`
- `../../README.md`
- `../../docs/release_facing_examples.md`
